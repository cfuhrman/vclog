VClog

VClog service generates changelogs from SCM commit messages.

Attributes

version[RW]

Current version of project.

format[RW]

Changelog format. Default is html. Supports html, xml, json, yaml, rdoc, markdown, and gnu.

type[RW]

Changelog layout type (log or rel). Default is log.

output[RW]

Output file to store changelog. The defualt is ‘log/vclog/changelog.{format}’.

rev[RW]

Show revision numbers (true/false)?

title[RW]

Some formats, such as rdoc, use a title field. Defaults to project title.

style[RW]

Some formats can reference an optional stylesheet (namely xml and html).
