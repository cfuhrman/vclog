DESCRIPTION

VCLog is a versitle cross-VCS/SCM changelog generator. It currently supports Git and Subversion.

SYNOPSIS

By default the format is a standard GNU text style. From a repository’s root directory try: 

  $ vclog

To generate an XML formatted changelog use: 

  $ vclog --xml

See ‘vclog —help’ for more options.
