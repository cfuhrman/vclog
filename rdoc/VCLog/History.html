Release History Class

A Release History is very similar to a ChangeLog. It differs in that it is divided into releases with version, release date and release note.

The release version, date and release note can be descerened from the underlying SCM by identifying the hard tag commits.

But we can also extract the release information from a release history file, if provided. And it’s release information should take precidence. ???
