DESCRIPTION

VClog is an cross-VCS/SCM changelog generator.

SYNOPSIS

VCLog generates changelogs. It supports a few different formats. The default format is standard GNU text style. From a repository’s root directory try: 

  $ vclog

To generate an XML formatted changelog use: 

  $ vclog --xml

See ‘vclog —help’ for more options.
